Extreme summer 2020 precipitation in Nepal

Evaluating GPM-era satellite precipitation over the Nepalese Himalaya

Two foundational evaluations of IMERG/TMPA against rain-gauge data across complex Himalayan terrain — quantifying accuracy in detecting extremes, drought, and elevation-dependent precipitation.

Outputs: Sharma et al., Earth and Space Science (2020); Sharma et al., Remote Sensing (2020).

Code & data

Where licensing permits, analysis code is released alongside each publication on GitHub (github.com/ss0293). Datasets follow the policies of the original data providers (NASA GPM, ECMWF, USGS, ESA, Australian Bureau of Meteorology).

Reproducibility checklist
  • Pinned Python environment via environment.yml (per project repo).
  • Notebooks executed top-to-bottom and stored with output cells stripped.
  • WRF namelist files, compile options and module lists included in each modelling repo.
  • DOI-pinned data references in every paper's data-availability statement.
